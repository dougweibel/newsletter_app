# Newsletter App Starter

Starter project for the Newsletter Information Gathering App.

## Stack
- Python 3.13
- PySide6
- SQLite

## First run
Activate your virtual environment, then from the project root run:

```bash
python -m src.app
```

## Current starter contents
- Main window with navigation buttons
- Members screen placeholder
- Events screen placeholder
- SQLite database initialization
- Initial schema for members, events, and member_events
